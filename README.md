# 2026 Summer Future Mobility Bootcamp

미래차 부트캠프 자료를 정리하는 저장소입니다.

## 목적

- 교안 원본을 과목별로 보관합니다.
- 퀴즈와 정답을 Markdown으로 정리합니다.
- 실습 코드, 실습 기록, 회고를 계속 추가합니다.
- 나중에 면접에서 설명할 수 있도록 핵심 개념과 경험을 한곳에 모읍니다.

## 구조

```text
미래차공학개론/
  README.md
  교안/              # PDF 교안 원본
  퀴즈/              # 퀴즈 정리본
  실습/              # 과목 관련 실습 기록
온디바이스-AI/
  README.md
  교안/              # PPTX 교안 원본
  퀴즈/              # 퀴즈 정리본
  실습/              # 과목 관련 실습 코드와 기록
```

## 현재 과목

- [미래차공학개론](미래차공학개론/README.md)
- [온디바이스 AI](온디바이스-AI/README.md)

## 업데이트 규칙

1. 새 교안은 해당 과목의 `교안/`에 원본 그대로 추가합니다.
2. 퀴즈는 해당 과목의 `퀴즈/`에 `.md`로 정리하고 정답을 포함합니다.
3. 실습 코드는 해당 과목의 `실습/` 아래에 날짜 또는 주제별 폴더로 분리합니다.
