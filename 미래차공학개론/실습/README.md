# 미래차공학개론 실습

오프라인 실습 코드와 실습 기록을 저장하는 공간입니다.

## 권장 구조

```text
실습/
  YYYY-MM-DD-topic/
    README.md   # 실습 목표, 실행 방법, 결과, 배운 점
    src/        # 코드
    data/       # 작은 샘플 데이터, 대용량 데이터는 링크만 기록
    outputs/    # 결과 이미지나 로그
```

## 실습 기록 템플릿

```md
# 실습 제목

## 목표

## 실행 방법

## 결과

## 배운 점

## 면접에서 설명할 포인트
```
