## 정리할 축

- [교안 자동 추출 텍스트](노트/extracted-text/)
- 미래차 산업이 기존 자동차 산업과 달라지는 지점
- 전동화, 자율주행, 연결성, 서비스화가 만드는 변화
- 제조공정과 안전/법규가 기술 채택에 주는 제약
- AAM, UAM, 버티포트처럼 새로운 모빌리티 인프라가 필요한 이유

## 면접 질문 후보

- 미래차의 핵심 역량은 기존 자동차와 어떻게 다른가?
- 커넥티비티가 차량 가치와 비즈니스 모델을 어떻게 바꾸는가?
- 모빌리티 안전과 법규가 기술 개발 방향에 어떤 영향을 주는가?
- AAM/UAM이 도입되려면 차량 기술 외에 어떤 인프라가 필요한가?
