# 미래차공학개론 퀴즈

현재 확인된 원본 파일 [raw-weekly-quiz.txt](raw-weekly-quiz.txt)는 **0바이트로 비어 있습니다.**

확인한 위치:

- `C:\성균관대\10_학교\3-1\여름_계절학기\미래차공학개론\주차별 퀴즈 단순모음.txt`
- `미래차공학개론/퀴즈/raw-weekly-quiz.txt`

즉, 미래차공학개론은 전범위 퀴즈 파일이 있어야 하는 자리와 파일명은 확인됐지만, 현재 디스크에 저장된 파일에는 문제 텍스트가 없습니다. 실제 퀴즈 추출본이 다시 저장되면 아래 형식으로 바로 Markdown 정리합니다.

## 정리 형식

```md
## 문제 N

문제:

선택지:

- A
- B
- C
- D

정답:

해설:
```

## 예정 파일

- `weekly-quiz.md`: 전범위 주차별 퀴즈 정리본
- `raw-weekly-quiz.txt`: 웹에서 추출한 원본 텍스트
