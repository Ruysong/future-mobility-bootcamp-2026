# 2일차 CAN FD 실습

2일차 실습 자료를 추가하기 위한 자리입니다.

현재 로컬 원본 폴더 `차량용통신 CAN(FD)/오프라인 2일차`에는 업로드할 파일이 없는 상태로 확인했습니다. 2일차 자료가 추가되면 아래 구조로 정리합니다.

## 추가 예정 구조

| 구분 | 설명 |
| --- | --- |
| 교안 | CAN FD 실습 PDF 또는 PPT |
| 실습 파일 | HEX, workspace, UI, capture file 등 |
| README | 실습 목표, 수행 내용, 결과, 면접 설명 포인트 |

## 정리 예정 개념

- CAN FD frame 구조
- Arbitration phase와 data phase
- BRS bit, ESI bit, FDF bit
- DLC와 최대 64 byte payload
- Classic CAN과 CAN FD의 속도/호환성 차이
