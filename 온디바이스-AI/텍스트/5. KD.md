# 4. Knowledge Distillation

## Concept of temperature

Recall softmax function:

```text
q_i = e^(z_i) / sum_j e^(z_j)
```

Softmax with temperature:

```text
q_i = e^(z_i / T) / sum_j e^(z_j / T)
```

## Example Table

| z | q [T=1] | q [T=10] |
| ---: | ---: | ---: |
| -1.1 | 0.007 | 0.171 |
| 1.4 | 0.087 | 0.219 |
| 3.7 | 0.880 | 0.276 |
| 0.1 | 0.024 | 0.193 |
| -3.0 | 0.001 | 0.141 |

## Temperature Effect

- Additional hyper-parameter: `T`
- Exposes more "dark knowledge"
- Probability curves shown:
  - `T=1`: sharper distribution
  - `T=5`: softer distribution
  - `T=10`: softest distribution

Source note: Adapted from "Machine Learning Hardware and Systems" Lecture (Fall 2024) by Prof. M. Abdelfattah (Cornell Tech).

## Matching output logits

## Diagram

- Input branches into teacher model and student model.
- Teacher Model
  - Layer 1 -> Layer 2 -> ... -> Layer N -> Logits
- Student Model
  - Layer 1 -> Layer 2 -> ... -> Layer N -> Logits
- Teacher logits and student logits connect to Distillation Loss.
- Student logits also connect to Classification Loss.

## Loss Options

Cross entropy loss:

```text
E(-p_t log p_s)
```

L2 loss:

```text
E(||p_t - p_s||_2^2)
```

## Matching intermediate features

# 4. Knowledge Distillation - What to Match?

## Matching intermediate attention maps

- Gradients of feature maps are used to characterize "attention" of DNNs.
  - The attention of a CNN feature map `x` is defined as `dL/dx`, where `L` is the learning objective.
  - Intuition: If `dL/dx_{i,j}` is large, a small perturbation at `i,j` will significantly impact the final output. As a result, the network is putting more attention on position `i,j`.

- Red annotation: Match intermediate attention maps `dL/dx`.

## Slide 14

# 4. Knowledge Distillation - What to Match?

## Matching sparsity patterns

- Intuition: the teacher and student networks should have similar sparsity patterns after the ReLU activation.
- A neuron is activated after ReLU if its value is larger than 0, denoted by the indicator function:

```text
rho(x) = 1[x > 0]
```

## Diagram

- Input branches into teacher and student paths.
- KD Loss terms match intermediate sparsity patterns between corresponding layers.
- Red annotation: Match intermediate sparsity pattern `rho(x) = 1[x > 0]`.
- Student logits connect to Classification Loss.
